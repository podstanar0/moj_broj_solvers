# moj broj solvers
moj broj game solvers implemented in multiple styles/paradigms.

about game: https://www.slagalica.tv/game/mojbroj

---

## OOP - Python 3
### usage: 
`python moj_broj_solver.py num1 num2, ... num6 -c targetNum`

## Functional - Haskell
### compiling:
`ghc moj_broj_solver.hs`

### usage:
`./moj_broj_solver`
