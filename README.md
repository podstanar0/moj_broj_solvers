# moj broj solvers
[moj broj](https://www.slagalica.tv/game/mojbroj) game solvers implemented in multiple styles/paradigms.

---

## OOP - Python 3
### usage: 
`python moj_broj_solver.py num1 num2, ... numN -c targetNum`

## Functional - Haskell
### usage:
compile using:

`ghc moj_broj_solver.hs`

then execute:

`./moj_broj_solver`
